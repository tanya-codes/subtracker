# ✦ SubTracker

> Track your subscriptions, monitor monthly costs & get billing reminders.

Built with pure HTML, CSS & JavaScript — no frameworks, no libraries, no setup needed.

---

## 🌸 About

I built SubTracker as my JavaScript subject project. The idea came from a real problem — I kept forgetting which subscriptions were renewing and how much I was actually spending every month. So I built a solution from scratch.

---

## ✨ Features

- ➕ Add, edit & delete subscriptions
- 💰 Auto-calculates monthly & yearly total cost
- 🔔 Billing reminders for renewals due today or tomorrow
- 🗂️ Filter by category — Streaming, Music, Productivity, Software
- 📅 "Due this week" counter on the dashboard
- 💾 All data saved in localStorage — no backend, no login needed

---

## 🛠️ Tech Stack

- **HTML5** — page structure
- **CSS3** — styling, dark theme, animations, CSS variables
- **JavaScript** — DOM manipulation, localStorage, date logic, array methods

---

## 💡 JavaScript Concepts Used

- DOM manipulation — dynamically creating cards with `innerHTML`
- `localStorage` — saving and loading data without a backend
- Array methods — `.map()` `.filter()` `.find()` for rendering and filtering
- Date object — calculating days until next billing date
- Event handling — all button clicks and form interactions
- Template literals — building HTML strings cleanly
